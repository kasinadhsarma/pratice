# 🧮 Basic Calculations

**Location:** [`basiccalculations/`](../basiccalculations/)

Simple procedural Python scripts. Each script takes two integers from user input, applies a standard arithmetic operator, and prints the result.

---

## Programs

| File | Operation | Formula |
| :--- | :--- | :--- |
| `addition.py` | Addition | `a + b` |
| `substraction.py` | Subtraction | `a - b` |
| `multiplication.py` | Multiplication | `a * b` |
| `division.py` | Division | `a / b` |

---

## Complexity

| Metric | Value |
| :--- | :--- |
| Time Complexity | $\mathcal{O}(1)$ — single arithmetic operation |
| Space Complexity | $\mathcal{O}(1)$ — two input variables, one output |

---

## How to Run

```bash
python ./basiccalculations/addition.py
python ./basiccalculations/division.py
```
